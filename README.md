# توثيق بوت نومية

## الميزات
- **ميزات متعددة تهدف لتحسين تجربة المستخدمين.**
- **التفاعل السريع والدعم للمستخدم بشكل فعال.**
- **تكامل مع واجهة برمجة تطبيقات Google Gemini لتوفير ميزات مبتكرة.**

## تعليمات التثبيت
1. قم بت clone من مستودع البوت:
   ```
   git clone https://github.com/ahmedwlaeed43-spec/bot.git
   ```
2. انتقل إلى مسار المشروع:
   ```
   cd bot
   ```
3. قم بتثبيت المتطلبات باستخدام بيثون:
   ```
   pip install -r requirements.txt
   ```

## دليل الاستخدام
- للتشغيل البوت، استخدم الأمر:
   ```
   python bot.py
   ```

## إعداد مفتاح API لبرنامج Telegram Bot
1. قم بفتح تطبيق Telegram وابحث عن `@BotFather`.
2. أنشئ بوتًا جديدًا واحتفظ بالمفتاح الناتج.

## إعداد مفتاح API لـ Google Gemini
1. اذهب إلى [Google Cloud Console](https://console.cloud.google.com/).
2. أنشئ مشروعًا جديدًا.
3. فعّل Google Gemini API.
4. احصل على مفتاح API واحتفظ به في ملف إعداد.

## ملاحظات
تأكد من أن جميع المتطلبات الضرورية قد تم تثبيتها وأنك قمت بإعداد جميع المفاتيح بشكل صحيح قبل تشغيل البوت.
